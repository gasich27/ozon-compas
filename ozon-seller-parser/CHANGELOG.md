# Changelog

Все заметные изменения в проекте документируются в этом файле.

## [2.0.0] — 2025-03-20

### Изменено
- Полная реструктуризация: модульная архитектура `src/`
- CLI точка входа `run.py` с командами `collect`, `scrape`, `all`
- Конфигурация через `dataclass` вместо глобальных переменных
- Профессиональная структура проекта

### Добавлено
- `src/config.py` — конфигурация через JSON с валидацией
- `src/browser.py` — инициализация WebDriver
- `src/collector.py` — Stage 1: скроллинг и сбор ссылок
- `src/scraper.py` — Stage 2: парсинг деталей товаров
- `config.example.json` — шаблон конфигурации
- Логирование в файл и консоль
- `.gitignore` для Python-проектов
- Лицензия MIT

### Удалено
- `stage1.py`, `stage2.py`, `main.py` — заменены модульной структурой
- Бинарные файлы (Chrome, chromedriver)
- Файлы данных (CSV, TXT, логи)
- Резервные копии и артефакты сборки

## [1.0.0] — 2025-01-16

### Добавлено
- Первая версия: `stage1.py` (сбор ссылок) + `stage2.py` (парсинг деталей)
- `main.py` — быстрый парсер одной страницы
- Обход антибот-защиты через `undetected-chromedriver`
- Инкрементальный сбор (не дублирует ссылки)
- Экспорт в CSV
