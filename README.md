# Ozon Seller Radar

Ozon Seller Radar — локальный веб-сервис для селлера Ozon. Сейчас это ранний доступ: интерфейс уже рабочий, но местами он ещё простой и слегка навайбкоденный.

Сервис помогает:

- синхронизировать свои товары через Ozon Seller API;
- запускать внешний парсер маркета по ссылке;
- загружать готовые CSV;
- хранить свои данные отдельно для каждого пользователя;
- сравнивать свой товар с товарами маркета;
- собирать отчёты и аналитику;
- работать через браузер без командной строки в обычном сценарии.

## Что уже есть

- регистрация и вход по email/паролю;
- личный кабинет пользователя;
- сохранение `OZON_CLIENT_ID` и `OZON_API_KEY` в интерфейсе;
- отдельные данные и наборы CSV для каждого пользователя;
- запуск внешнего парсера как отдельного процесса;
- страница парсинга с прогрессом и отменой;
- загрузка и просмотр CSV;
- сравнение товара селлера с рынком;
- отчёты в `TXT`, `JSON` и `XLSX`;
- простой стеклянный интерфейс с мягким фоном и анимациями.

## Как это устроено

- Данные хранятся в SQLite.
- Для каждого пользователя отдельно сохраняются:
  - аккаунт и пароль;
  - ключи Ozon Seller API;
  - товары селлера;
  - загруженные CSV;
  - результаты парсинга;
  - отчёты и сравнения.
- Внешний парсер лежит отдельно в папке `ozon-seller-parser/`.
- Ozon Seller Radar не меняет внутреннюю логику внешнего парсера, а просто запускает его как subprocess.

## Установка

Требования:

- Python 3.11+;
- Google Chrome свежей версии;
- доступ к Ozon Seller API, если нужен режим своих товаров;
- установленный внешний парсер в папке `ozon-seller-parser/`.

Установка зависимостей:

```bash
python -m pip install -r requirements.txt
```

Если планируете использовать внешний парсер отдельно, проверьте его зависимости в `ozon-seller-parser/requirements.txt`.


## Запуск

Запуск веб-приложения:

```bash
python web.py
```

После этого откройте:

```text
http://127.0.0.1:8000
```

## Как пользоваться

1. Зарегистрируйтесь или войдите.
2. На странице `Мои товары` сохраните `OZON_CLIENT_ID` и `OZON_API_KEY`.
3. Синхронизируйте товары селлера.
4. На странице `Парсинг товаров`:
   - запустите внешний парсер по ссылке;
   - либо загрузите готовый CSV;
   - дождитесь завершения;
   - при необходимости удалите лишние строки и сохраните итог.
5. На странице `Сравнение` выберите свой товар и набор данных маркета.

Во время парсинга:

- может открыться окно браузера;
- его лучше не закрывать до завершения;
- парсинг может занять несколько минут, иногда до 10–15 минут;
- после завершения CSV автоматически уходит в аналитику.

## Отчёты и экспорт

Система создаёт отчёты для:

- товаров селлера;
- товаров маркета;
- сравнения товара селлера с рынком.

Отчёты можно скачать в `TXT`, `JSON` и `XLSX`.

## Внешний парсер

Внешний парсер уже существует отдельно в `ozon-seller-parser/`.

- Он запускается как отдельный процесс.
- Он использует свой `config.json`.
- Он может открывать окно Chrome.
- Его окно лучше не закрывать до завершения работы.
- CSV парсера содержит колонки:
  - `Название`
  - `Ссылка на товар`
  - `Артикул`
  - `Цена`
  - `Количество отзывов`
  - `Средний отзыв`

Если внешний парсер требует ручной настройки, ChromeDriver или отдельные параметры, настраивайте его по его README. Ozon Seller Radar не переписывает парсер и не меняет его внутреннюю логику.
